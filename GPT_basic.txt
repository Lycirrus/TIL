UI : 사용자와 소프트웨어 간의 상호 작용 시스템
-> ATM의 선택 화면
-> 스마트폰에서 사용하는 제스처, 푸시 알림

API(application programming interface)
: 두 소프트웨어가 서로 통신할 수 있게 하는 매커니즘
->문제 : 소프트웨어 간의 언어가 다름 => 규격 정의

API예시
1. 특정 사이트에서 기존 소셜 미디어 계정으로 로그인
2. 기상청의 날씨 데이터를 기기 DB에 제공
-> 데이터 제공자가 제작한 문서에 맞춰 요청해야 함
 : 새로고침마다 요청을 보내고 있음

API키는 제공 서버가 요청인에게 권한을 허락하기 위해 존재
