  #### 문자열을 input하는 방법간 차이
  - `list(input())` 
    ```python
    # 입력 : 'abc'
    print(list(input()))

    # 출력 : ['a', 'b', 'c']
    ```
    `list`로 감쌀 경우, 문자의 철자 하나하나 나누어 리스트 형태로 들어온다.
  - `input()`
    ```python
    # 입력 : 'abc'
    print(input())

    # 출력 : abc
    ```
    `input()`만 사용할 경우, 입력한 문자열 그래도 들어온다.
